## What is Vinext?

Vinext is a Vite plugin that reimplements the Next.js API surface for Cloudflare Workers. It supports App Router, RSC, and file-based routing.

## Why Content-Driven?

Content-driven sites separate content from presentation. MDX files hold structured data in frontmatter and prose in the body, making it easy to build blogs, project showcases, and documentation.

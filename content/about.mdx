This site is built with **Vinext** and deployed to **Cloudflare Workers**.

It serves as a personal space for blog posts, project showcases, and notes. The content layer uses MDX with frontmatter validation to ensure every entry is well-structured before it reaches the build.

## Tech Stack

- **Framework**: Vinext (Vite + React App Router)
- **Runtime**: Cloudflare Workers
- **Content**: MDX with build-time validation
- **Styling**: Semantic CSS tokens

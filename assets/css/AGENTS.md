# CSS KNOWLEDGE BASE

## OVERVIEW

`main.css` is the global stylesheet for the whole site.

## WHERE TO LOOK

- `main.css` - layout, typography, utility-style classes, and page styling

## CONVENTIONS

- Edit `main.css` directly; there is no CSS build pipeline beyond Vinext/Vite.
- Match the existing hand-written utility-style naming.

## ANTI-PATTERNS

- Do NOT split styles into a framework-specific structure.
- Do NOT assume CSS modules or Tailwind are available.

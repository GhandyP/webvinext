# ASSETS KNOWLEDGE BASE

## OVERVIEW

Static source assets live here. In this repo that currently means the hand-written global stylesheet under `css/`.

## WHERE TO LOOK

- `css/main.css` - global site styles used by `_app.tsx`

## CONVENTIONS

- Keep assets source-controlled and human-edited.
- Favor updating existing CSS patterns over introducing a framework.

## ANTI-PATTERNS

- Do NOT add generated build output here.
- Do NOT introduce Tailwind or another CSS framework without changing project-level guidance first.

# Integraciones y Servicios Externos

Opciones para integrar servicios externos en el proyecto Vinext.

## Analytics

### Cloudflare Web Analytics
- Configuración desde Cloudflare Dashboard
- Privacy-first, sin cookies
- Integración nativa con Workers

### Alternativas
- Plausible: open source, privacy-focused
- Umami: self-hosted, privacy-focused

## Comentarios

### Giscus
- Usa GitHub Discussions como backend
- Gratuito para proyectos open source

### Utterances
- Usa GitHub Issues como backend
- Más simple que Giscus

## Búsqueda

### Pagefind
- Indexación estática
- Excelente performance
- Se genera en build time

### Fuse.js
- Cliente-side
- Ligero y rápido

# BASE DE CONOCIMIENTO - OPERACION

## OVERVIEW

Aqui vive la documentacion operativa: setup local, entorno, despliegue y workflow de agentes.

## WHERE TO LOOK

- `SETUP.md` - preparacion del entorno
- `ENVIRONMENT.md` - variables y contexto de runtime
- `DEPLOYMENT.md` - flujo de despliegue
- `OPERATIONS.md` - runbook
- `OPENCODE_WORKFLOW.md` - reglas de trabajo con agentes

## CONVENTIONS

- Mantener pasos operativos ejecutables y concretos.
- Reflejar el workflow real basado en Bun y Vinext.

## ANTI-PATTERNS

- No dejar comandos obsoletos tras cambios de tooling.
- No documentar despliegues sin verificar el flujo actual.
